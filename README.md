# Transit Empire (Prototype)

A starter Python + pygame prototype for **Transit Empire**.

## Features
- Stations spawn over time.
- Passengers spawn with random destinations.
- Visualizes stations and waiting passengers.

## Run
```bash
pip install pygame
python main.py
```
